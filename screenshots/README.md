# Screenshots Directory

Add your screenshots here with the following names:

- `login.png` - Login page screenshot
- `welcome.png` - Welcome/Dashboard page with book list
- `create.png` - Add new book page
- `details.png` - Book details page
- `edit.png` - Edit book page

## How to Add Screenshots

1. Take screenshots of each page (use Cmd+Shift+4 on Mac to capture specific areas)
2. Save them in this `screenshots/` directory
3. Name them exactly as listed above
4. The README.md will automatically reference them

## Recommended Screenshot Sizes

- Optimal width: 1200-1600px
- Format: PNG or JPG
- Make sure the screenshots showcase the design nicely!
